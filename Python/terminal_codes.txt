"ni" - new file Eg: ni example.txt
"del" - delete file Eg: del example.txt 
"cd.." - Get back to previous directory

"git add ." - stage all changes
"git add <fileName1> <fileName2>" - stage one file 
"git reset ." - unstage all files 

"git commit -m "_message_"" - Commit all.
"git commit -m "_message_" -- path/<file.txt>" - Commit specific file 
"git revert HEAD" - undo without deleting the history

"git push" - push to Main
"git push -u origin <branch_name>" - When pushing a branch for the first time, use the -u (or --set-upstream) flag. 
This links your local branch to the remote one so you can just use git push in the future.
"git checkout -b <branch_name>" - create branch and switch to it in one step.
